# video-transcriber

Created by OKIHIRO — AI自動化ワークフローを130個以上開発・運用しています。

`video-transcriber` は、動画・音声ファイルからSRT字幕とプレーンテキストの文字起こしを生成するAIエージェント向けスキルです。

## 何ができるか

- mp4、mp3、m4a、wav、mkv、mov の文字起こし
- SRT字幕ファイルの生成
- タイムコードなしの `transcript.txt` 生成
- 日本語パス回避のための一時workspace処理

## 注意

- NVIDIA GPU + CUDA 12.x 推奨です。
- 外部APIキーは不要です。
- 配布物に音声・動画ファイルは含めません。
